Show post view and comment counts on frontend user dashboard.

== Description ==

> #### This add-on is for [**WP User Frontend**](https://wordpress.org/plugins/wp-user-frontend/) plugin. And uses [**WP-PostViews**](https://wordpress.org/plugins/wp-postviews/) plugin for post view count.

= Features:  =

* Show post view count on frontend user dashboard page
* Show comment count on frontend user dashboard page

== Installation ==

1. Install [WP-PostViews](https://wordpress.org/plugins/wp-postviews/installation/) Plugin.

2. Then install this plugin.

3. Check your dashboard page and post view and comment counts will there. No configuration required.
